# mapm-to-critic
This small script ingests the [Miqra-al-pi-haMasorah](https://github.com/marcstober/miqra-data/tree/master) dataset
into [critic](https://github.com/curatorsigma/critic) as the base corpus.
It automatically calculates the Masoretic versification along the way and inserts this.

You may want to use this short binary as a starting point to ingest your own base corpus into critic.
